- To install kubesec plugin, Run
  <details>

  ```
  wget https://github.com/controlplaneio/kubesec/releases/download/v2.11.0/kubesec_linux_amd64.tar.gz

  tar -xvf  kubesec_linux_amd64.tar.gz

  mv kubesec /usr/bin/
  ```
 </details>

- **`Bash`** format is NOT supported by kubesec.
